ES6+ 문법 예제

브라우저 콘솔(F12)을 열어서 결과를 확인하세요!

1. let과 const

let (변수 재할당 가능)

let name = "홍길동"; name = "김철수"; // 재할당 가능 console.log(name); // "김철수"

const (상수, 재할당 불가)

const PI = 3.14159; // PI = 3.14; // 에러 발생! console.log(PI); // 3.14159

2. 화살표 함수 (Arrow Function)

기본 문법

// 기존 함수 function add(a, b) { return a + b; } // 화살표 함수 const add = (a, b) => { return a + b; } // 한 줄일 때 return 생략 가능 const add = (a, b) => a + b;

3. 템플릿 리터럴 (Template Literal)

문자열 연결

const name = "홍길동"; const age = 25; // 기존 방식 const message1 = "이름: " + name + ", 나이: " + age; // 템플릿 리터럴 const message2 = `이름: ${name}, 나이: ${age}`;

4. 기본 매개변수 (Default Parameters)

함수 매개변수 기본값

function greet(name = "손님") { return `안녕하세요, ${name}님!`; } greet(); // "안녕하세요, 손님님!" greet("홍길동"); // "안녕하세요, 홍길동님!"

5. 구조 분해 할당 (Destructuring)

배열 구조 분해

const colors = ["빨강", "파랑", "노랑"]; const [red, blue, yellow] = colors; console.log(red); // "빨강" console.log(blue); // "파랑"

객체 구조 분해

const person = { name: "홍길동", age: 25, city: "서울" }; const { name, age } = person; console.log(name); // "홍길동" console.log(age); // 25

6. 스프레드 연산자 (Spread Operator)

배열 복사 및 합치기

const arr1 = [1, 2, 3]; const arr2 = [4, 5, 6]; // 배열 복사 const arr3 = [...arr1]; // [1, 2, 3] // 배열 합치기 const arr4 = [...arr1, ...arr2]; // [1, 2, 3, 4, 5, 6]

7. 삼항 연산자 (Ternary Operator)

조건부 값 할당

const age = 20; const status = age >= 18 ? "성인" : "미성년자"; console.log(status); // "성인"

8. 옵셔널 체이닝 (Optional Chaining)

안전한 속성 접근

const user = { name: "홍길동", address: { city: "서울" } }; // 기존 방식 (에러 발생 가능) const city1 = user.address.city; // user.address가 null이면 에러 // 옵셔널 체이닝 (안전) const city2 = user.address?.city; // undefined 반환 (에러 없음)
