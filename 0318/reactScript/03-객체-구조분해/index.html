객체와 구조 분해 예제

React에서 props를 다룰 때 필수적인 개념입니다!

1. 객체 기본 문법

객체 생성 및 접근

const person = { name: "홍길동", age: 25, city: "서울" }; console.log(person.name); // "홍길동" console.log(person["age"]); // 25

2. 객체 구조 분해 (Destructuring)

기본 구조 분해

const person = { name: "홍길동", age: 25, city: "서울" }; const { name, age } = person; console.log(name); // "홍길동" console.log(age); // 25

3. 기본값 설정

구조 분해 시 기본값

const person = { name: "홍길동" }; const { name, age = 0, city = "미정" } = person; console.log(age); // 0 (기본값) console.log(city); // "미정" (기본값)

4. 변수명 변경

다른 이름으로 추출

const person = { name: "홍길동", age: 25 }; const { name: userName, age: userAge } = person; console.log(userName); // "홍길동" console.log(userAge); // 25

5. 중첩 객체 구조 분해

객체 안의 객체 분해

const user = { name: "홍길동", address: { city: "서울", district: "강남구" } }; const { name, address: { city, district } } = user; console.log(city); // "서울" console.log(district); // "강남구"

6. 함수 매개변수에서 구조 분해

React에서 자주 사용하는 패턴

// 기존 방식 function greet(person) { return `안녕하세요, ${person.name}님!`; } // 구조 분해 사용 function greet({ name, age }) { return `안녕하세요, ${name}님! (${age}세)`; } greet({ name: "홍길동", age: 25 });

7. 나머지 속성 (Rest Properties)

나머지 속성을 하나의 객체로

const person = { name: "홍길동", age: 25, city: "서울", job: "개발자" }; const { name, ...rest } = person; console.log(rest); // { age: 25, city: "서울", job: "개발자" }

8. 객체 스프레드 연산자

객체 복사 및 병합

const person1 = { name: "홍길동", age: 25 }; const person2 = { ...person1, city: "서울" }; // 결과: { name: "홍길동", age: 25, city: "서울" } const person3 = { ...person1, age: 30 }; // 결과: { name: "홍길동", age: 30 } (age 덮어쓰기)

9. React에서 사용하는 패턴

Props 구조 분해

// React 컴포넌트 예시 function UserCard({ name, age, email }) { return ( <div> <h3>{name}</h3> <p>나이: {age}</p> <p>이메일: {email}</p> </div> ); } // 사용 <UserCard name="홍길동" age={25} email="hong@test.com" />
