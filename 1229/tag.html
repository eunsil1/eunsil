태그란 무엇인가?

태그는 주로 html에서 사용되며, 웹 문서에서 “이 부분이 무엇인지”를 알려주는 표시(표식) 이다.

태그의 기본구조는?

기본구조는 꺾쇠로 열고 닫는 형태로 안쪽에 내용을 적는다.

태그의 분류

블럭요소란?

  1. 한줄 전체를 차지한다.
  2. 항상 줄바꿈이 발생
  3. 넓이(width), 높이(height) 설정가능
  4. margin(바깥여백), padding(안여백) 설정 가능
  5. 블럭요소 안에 인라인 요소 포함 가능(블럭요소도 포함 가능)
  6. 레이아웃 구성에 주로 사용

블럭요소의 종류

대표적인 블럭요소: div, p, h1~h6, ul, ol, li, br, pre, blockquote, address

인라인요소란?

글자 관련 인라인 태그이며, 텍스트 일부 강조에 쓰임

인라인요소의 종류?

문단/제목태그, 텍스트 강조태그, 인용/출처 태그, 기타 텍스트 관련 태그가 있다.

제목태그

헤딩태그라고 불리며, h1~h6 태그로 사용 / 계층 구조를 나타내는 제목태그이다.

문단태그

p태그라고 불리며, 일반적인 텍스트 문단을 나타낸다.

텍스트 태그

mark, a, strong, em, b, i, small, del, ins, u, sup, sub이 있다.

이미지태그

img태그로 사용되며, 이미지를 삽입할때 사용한다.
img src는 이미지 경로를 나타냄(src는 source의 약자)

강조태그

  • strong(강한강조 very important)과 em(약한강조 important)태그로 사용되며, 텍스트를 강조할때 사용한다.
  • b와 i태그도 있으나 잘 사용하지 않는다.(접근성 때문에 잘 사용하지 않음)

인용태그

blockquote태그로 사용되며, 인용문을 나타내는 블럭요소이다.

목록태그

ul, ol, li태그로 사용되며, 순서없는 목록과 순서있는 목록을 나타낸다.

구분선태그

hr태그로 사용되며, 주로 내용의 구분을 나타낼때 사용한다.(수평선으로 표시됨)

주소태그

address태그로 사용되며, 작성자나 소유자의 연락처 정보를 나타낼때 사용한다.(웹접근성 때문에 사용)

시맨틱태그

의미를 가진 태그로, 콘텐츠의 역할을 명확히 표현한다.

    주요 시맨틱태그
  • header: 문서나 섹션의 머리글을 나타낸다.
  • nav: 내비게이션 링크를 포함한다.
  • main: 문서의 주요 콘텐츠를 나타낸다.
  • section: 문서의 일반적인 구획을 나타낸다.
  • article: 독립적인 콘텐츠를 나타낸다.
  • aside: 주요 콘텐츠와 간접적으로 관련된 내용을 나타낸다.
  • footer: 문서나 섹션의 바닥글을 나타낸다.
